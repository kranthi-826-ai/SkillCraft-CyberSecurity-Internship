# Password Strength Checker

## Objective
To check the strength of a password using Python.

## Description
This project evaluates a password based on length, uppercase letters, lowercase letters, digits, and special characters.

## Features
- Detects weak passwords
- Detects medium passwords
- Detects strong passwords

## Tools Used
- Python 3
- Kali Linux
