# Image Encryption Tool

## Objective
To encrypt and decrypt image files using Python.

## Description
This project encrypts image files using XOR operations and decrypts them using the same key.

## Features
- Encrypt image files
- Decrypt image files
- XOR-based encryption

## Tools Used
- Python 3
- Kali Linux
